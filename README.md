# HEX_DEC_calculator

### HEX / DEC 기능 

1. HEX (A, B, C, D , E)에 마우스를 올리면 색 바뀜.
2. HEX or DEC 선택 할 때 색 변경으로 현재모드 표시.
3. HEX <-> DEC 변환가능.
4. 그 외 기본적인 계산가능.

**iPhone_Style_Calculator**
